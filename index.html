No Mapgen
for Minetest

Generate a world with only air.

Download GitHub Minetest CoRNeRNoTe

Features

  • Disables all mapgen, creating a void world.
  • Creates aliases to replace all mapgen aliases with air (you will need something like worldedit to place the 1st block).

Docs

Requirements

  • Minetest >=0.4.3

Installation

Download and uncompress the latest version, and move the the minetest-no_mapgen-master folder into your minetest/mods folder.
